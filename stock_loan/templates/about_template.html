About

I worked in finance for the past five years, with my last year or so doing a decent amount of short selling. One of the resources I used quite a lot was Interactive Broker's Stock Loan Availability Database especially for hard to borrow stocks.

But the format was terrible (CSV files) and and overall a pain to use. So, after taking Udacity's Full Stack Nanodegree I figured this database would be a fun place to start building a working (maybe even useful) web app.

Built in Python, I used the Flask framework and SQLAlchemy/SQLite for the user database and Postgresql for the borrow database. Performance is ok thanks to caching for everything but historical reports. Security should be decent thanks to Flask-Login and apache2 SSL (I will get around to purchasing a domain and getting a SSL certificate if people actually use this.

You can reach me at cameron.mochrie@gmail.com. The source for I Borrow Desk is available at GitHub
